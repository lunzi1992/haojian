## 项目

### [BabyTrip](./BabyTrip) - 宝宝出行助手 🧒👶

> 一个基于 Swift + SwiftUI 的原生 iOS + watchOS 应用，帮助家长判断当前天气环境是否适合带宝宝外出。

**功能特点：**
- 📱 支持 iOS 17+ / watchOS 10+
- 🌤 根据宝宝月龄智能调整评估标准
- 🧮 温度/紫外线/空气质量/风速 四维综合评估
- 🔔 恶劣天气预警提醒
- 👔 智能穿衣建议
- 🗺️ 基于位置的实时天气数据
- ⚙️ 自定义偏好设置
- 🔒 完全本地存储，无需登录，保护隐私

**技术栈：**
- Swift + SwiftUI
- QWeather API (和风天气)
- CoreLocation
- UserNotifications
- WatchConnectivity
